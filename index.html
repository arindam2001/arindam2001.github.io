About

Early-career computer science engineer and researcher focused on systems security, secure boot, trusted execution, and backend engineering.

I’m currently pursuing an M.Tech in Cyber Security at IIT Delhi, where my thesis focuses on Secure Boot and Trusted Logging for Automotive ECUs using HSM & TEE. I design security stacks that protect embedded platforms against firmware tampering, unauthorized modification, and runtime threats.

I enjoy working across the boundary of systems + security — kernel-level enhancements, cryptographic protocols, secure communications, and performance-aware networking. I also have hands-on experience building backend systems in Java/Spring.

My recent work includes privacy-preserving cloud integrity auditing, Android and web security testing, reliable transport protocols, SDN routing optimization, and blockchain-based applications.
